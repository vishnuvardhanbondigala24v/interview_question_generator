##    AI Interview Studio

**AI Interview Studio** is a voice-enabled, AI-powered interview preparation platform built with Streamlit. It generates personalized interview questions based on your resume, job role, and desired difficulty level, and evaluates your responses in real time. Whether you're preparing for a technical, behavioral, or HR interview — this app helps you practice smarter.



## 🔗 Live Demo

[Click here to try AI Interview Studio](https://your-streamlit-app-url.streamlit.app)

---

### 🚀 Key Features

- **   Personalized Question Generation**  
  Generate interview questions tailored to your job role, difficulty level, and selected categories.

- **📄 Resume & Job Description Analysis**  
  Upload your resume and paste a job description to extract relevant skills and customize your interview prep.

- **🎙️ Voice Input Support**  
  Answer questions using your microphone with built-in speech-to-text functionality.

- **✅ Real-Time Answer Evaluation**  
  Receive instant feedback and scoring for each answer using AI-powered evaluation.

- **📊 Visual Insights**  
  View charts showing question breakdown and extracted skills from your resume and job description.

- **📥 Export Questions**  
  Download your generated questions as TXT or CSV for offline practice.

###  Technologies Used

###  Installation

```bash
# Clone the repository
git clone https://github.com/your-username/ai-interview-studio.git
cd ai-interview-studio

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
